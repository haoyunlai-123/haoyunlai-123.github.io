如何把git上的项目拉到本地F:\JavaXiangMu\tianjixuetang\tianjixuetang下？这两个项目

要把 **GitHub** 和 **Gitee** 上的两个仓库拉到本地的：

```
F:\JavaXiangMu\tianjixuetang\tianjixuetang
```

目录下，你只需要执行 **git clone + 自己想放的目录名** 即可。

下面给你写得非常清楚，照着做就行。

------

# ✅ 一、先进入目标文件夹

打开 **终端（PowerShell 或 CMD）**，执行：

```bash
cd /d F:\JavaXiangMu\tianjixuetang\tianjixuetang
```

确认你现在就在这个目录下：

```bash
pwd
```

输出应该是：

```
F:\JavaXiangMu\tianjixuetang\tianjixuetang
```

------

# ✅ 二、克隆 GitHub 项目

GitHub 项目地址：

```
https://github.com/shuhongfan/tjxt.git
```

克隆命令（克隆到当前目录下，并且目录名叫 `tjxt-github`，避免和 Gitee 冲突）：

```bash
git clone https://github.com/shuhongfan/tjxt.git tjxt-github
```

执行完后会出现：

```
F:\JavaXiangMu\tianjixuetang\tianjixuetang\tjxt-github
```

------

# ✅ 三、克隆 Gitee 项目

Gitee 项目地址：

```
https://gitee.com/fenny_ye/tjxt.git
```

克隆命令（放到目录名为 `tjxt-gitee`）：

```bash
git clone https://gitee.com/fenny_ye/tjxt.git tjxt-gitee
```

执行后：

```
F:\JavaXiangMu\tianjixuetang\tianjixuetang\tjxt-gitee
```

------

# 📌 最终目录结构应该是：

```
F:\JavaXiangMu\
 └─ tianjixuetang\
     └─ tianjixuetang\
         ├─ tjxt-github\
         └─ tjxt-gitee\
```

这样两个仓库分开，不会冲突，IDEA 打开哪个就选哪个。

------

# ❤️ 如果你希望目录名保持一致怎么办？

比如你只想要：

```
.../tianjixuetang/tjxt
```

那你只克隆一个即可：

```bash
git clone https://github.com/shuhongfan/tjxt.git tjxt
```

但两个仓库内容可能不同，不建议这样，会互相覆盖。

------

# 📢 如果你要推送代码回 GitHub / Gitee

记得配置：

```bash
git config --global user.name "你的名字"
git config --global user.email "你的邮箱"
```

然后用 HTTPS 或 SSH 登录。
